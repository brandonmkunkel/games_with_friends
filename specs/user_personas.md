# Personas

## Host

The host will be in control of the game.

The host:
- Will run from a computer connected to a TV. 
- Will need to have one screen (the host screen) for controlling the flow of the game/scores/players.
- Will need to have a second screen (the display screen) for displaying the current question/answers/scores. 
- Will need to be able to restart the game from the host screen.
- May need to end the game early. 

## Player

The player participating in the trivia games. 

A player:
- May be on their phones or on a computer.
- Will use their phone or computer to answer questions and buzz in.
- Will not join teams, all plyers will play individually.
- Will need to be able to see the current question on their phone
- Will need to see their current score
